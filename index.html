Spelling Game

A front-end based game to teach children how to spell the names of various animals. The game displays the word with one randomly chosen letter ommitted. The child can view a picture of the animal and hear the pronounciation of the word.

When the correct letter is chosen, the child is presented with the Spanish, French and Mandarin translation of the word along with pronounciations. An exerpt from the Wikipedia article about the animal is displayed with a link back to the full article.

Technologies: Javascript, HTML5, CSS3, jQuery, Twitter Bootstrap, AJAX, MediaWiki API, Google Translate API, WebSpeech API
